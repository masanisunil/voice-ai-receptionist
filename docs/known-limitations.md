# Known Limitations

- Live Retell phone number is not provisioned in this local repo.
- Cliniko credentials are not included; mock PMS is used.
- NU Hospitals live slot inventory is not public, so seed availability is synthetic.
- Local eval cannot measure true ASR and TTS latency without Retell call traces.
- Hindi voice quality depends on selected Retell voice and ASR configuration.
- Pricing/currency is not implemented because the seeded clinic flow does not mention prices.
